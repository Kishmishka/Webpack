# WEBPACK
## Команды

### Инициализация
```shell
npm init
npm i
```
### Запуск сервера для разработки
```shell
npm run start
```

### Сборка проекта без оптимизации
```shell
npm run dev
```

### Сборка проекта с оптимизацией
```shell
npm run prod
```

### Очистка папки dist
```shell
npm run clear
```
